# Selected biostatistics coursework, 2019-2020

#### Longitudinal and Multilevel Data Analysis, Spring 2020
Topics include linear mixed effects models, hierarchical models, cross-classified multilevel models, 
generalized and nonlinear mixed effects models, marginal models via generalized estimating equations (GEE), 
missing data in longitudinal studies, etc.

#### Survival Analysis, Spring 2020
Topics include include construction and analysis of failure rates, survival curves, 
hypothesis tests for comparing survival curves, parametric models, semiparametric models for the analysis of time-to-failure data including the Cox proportional hazards model, etc.

#### Theory of Statistics, Spring 2020
Topics include hypothesis testing, estimation, confidenceintervals, maximum likelihood, Bayesian analysis, 
asymptotic approximations, computer simulation, generalized linear models, etc.

#### Advanced Statistical Programming in SAS and R, Fall 2019
Topics include SAS macros, proc SQL, simulation in R, etc.

#### Advanced Linear model, Fall 2019 
Topics include linear regression, estimation, hypothesis testing, regression diagnostics, analysis of variance, adjusting for covariates, transformations, missing data, and generalized linear models.
